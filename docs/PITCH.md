# Pitch La Caverne aux 40 Voleurs — Qwen Cloud Global AI Hackathon 2026

## Hook (15 s)

**Ali Baba avait 40 voleurs. Vous, vous avez 40 experts Qwen.**  
Un Génie les dirige. Une Balance prouve le gain. Une Lampe apprend qui vous êtes.

## Problème (30 s)

Un seul LLM fort = cher et parfois surcoté.  
Un LLM faible = insuffisant.  
Comment obtenir **plus de qualité pour moins de tokens** sans réécrire ses prompts à chaque fois ?

## Solution (45 s)

**La Caverne aux 40 Voleurs** est un orchestrateur MoE multi-agents natif Qwen Cloud :

- **Voleurs** : experts spécialisés (code, stratégie, UX, critique…).
- **Génie** : assemble les meilleurs experts par routing embedding × perf historique.
- **Balance du Marchand** : compare coût Caverne vs agent solo en temps réel.
- **Arène du Sabre** : juge Qwen-max tranche qualité bande vs solo.
- **La Lampe** : avatar compagnon né de vos échanges, qui fusionne avec votre style.
- **Le Connecteur** : un savoir partagé par tous, de 1B à 1T tokens.
- **Fable 5 / OpenRouter** : quand Qwen n'a pas le modèle idéal, la Caverne le trouve ailleurs.

## Démonstration (60 s)

1. Créer 4 Voleurs dans **Le Camp**.
2. Forger un Génie avec routing strategy + ML + embedding model dans **Le Génie**.
3. Poser une question complexe → fragments + fusion.
4. Lancer un duel dans **L'Arène du Sabre**.
5. Voir le gain dans **Les Trésors** / **Balance**.
6. Taper `shazaam` dans **L'Atelier** pour révéler **La Lampe**.

## Différenciation Qwen (30 s)

- Provider **Qwen Cloud par défaut** : `qwen-turbo`, `qwen-plus`, `qwen-max`, `qwen-coder-plus`, `qwen-vl-plus`, `text-embedding-v3`, `cosyvoice-v1`, `wanx2.1`.
- **80 couches** couvrant persistance, humanité, KB, profilage, tools, agents, vision, connaissance partagée.
- **Zero dépendance lourde** : Node ESM natif + React + Vite.

## Impact (20 s)

- Réduction de coût mesurable.
- Meilleure qualité grâce au vote des experts.
- Personnalisation via l'avatar compagnon.
- Savoir collectif croissant.

## Appel (10 s)

**Sésame, ouvre-toi.**  
La Caverne est prête. Le Génie attend. La Lampe n'attend que votre voix.
